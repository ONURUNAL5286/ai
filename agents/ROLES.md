# Agent Roles

## 1. CEO / Human Supervisor

Sahibi sensin. Musteri secimi, kapsam onayi, butce, production deploy ve teslim onayi sende kalir.

## 2. Product Manager Agent

Gorevleri:
- Sprint hedefini netlestirir.
- User story yazar.
- Acceptance criteria belirler.
- Tasklari onceliklendirir.
- Sprint sonunda tamamlanan/kalan isleri raporlar.

Basari kriteri:
- Her task test edilebilir ve net olmalidir.

## 3. Market Research Agent

Gorevleri:
- Turkiye pazarinda eksik yazilim urunlerini arastirir.
- KOBI, e-ticaret, uretim, saglik turizmi, lojistik, hukuk, finans ve egitim dikeylerini tarar.
- Rakipleri, fiyatlari ve musteri acilarini cikarir.
- Yeni sprint fikirleri uretir.

Basari kriteri:
- Her fikir hedef musteri, aci, MVP kapsami ve gelir modeliyle gelmelidir.

## 4. UX/UI Agent

Gorevleri:
- Kullanici akislarini cizer.
- Wireframe ve ekran listesini hazirlar.
- Frontend agenta tasarim kararlarini verir.

Basari kriteri:
- Uygulama ilk acilista kullanilabilir olmalidir; landing page degil, gercek urun ekrani onceliklidir.

## 5. Frontend Agent

Gorevleri:
- UI implementasyonu.
- Responsive davranis.
- Formlar, tablolar, paneller.
- Frontend testleri.

Basari kriteri:
- Mobil ve desktop gorunum bozulmamalidir.

## 6. Backend Agent

Gorevleri:
- API endpointleri.
- Yetkilendirme.
- Veritabani modelleri.
- Is kurallari.
- Backend testleri.

Basari kriteri:
- API hatalari anlamli donmeli, testler gecmelidir.

## 7. QA Agent

Gorevleri:
- Acceptance criteria kontrolu.
- Playwright veya manuel test senaryolari.
- Demo URL uzerinden smoke test.
- Bug issue'lari acma.

Basari kriteri:
- Kritik akislarda calismayan ozellik kalmamalidir.

## 8. DevOps Agent

Gorevleri:
- GitHub Actions.
- Environment variables.
- Deploy ayarlari.
- Build ve migration kontrolu.

Basari kriteri:
- Yeni bir geliştirici README ile projeyi ayaga kaldirabilmelidir.

## 9. Security Reviewer Agent

Gorevleri:
- Secret leak kontrolu.
- Dependency vulnerability taramasi.
- Auth ve veri erisim kontrolu.
- Kisisel veri risklerini isaretleme.

Basari kriteri:
- Production'a cikacak sistemde temel guvenlik acigi olmamalidir.
