## Ornek

```text
SPRINT BASLAT

Proje adi: KOBI Teklif Takip

Hedef: Kucuk isletmelerin tekliflerini hazirlayip takip edebilecegi web uygulamasi.

Kullanici tipi: 5-50 calisani olan hizmet firmalari.

Ana ozellikler:
1. Musteri kaydi
2. Teklif olusturma
3. Teklif durum takibi

Olmazsa olmazlar:
1. Mobil uyumlu panel
2. PDF teklif cikarma
3. Basit login sistemi

Teknik tercih:
- Frontend: Next.js
- Backend: Next.js API veya NestJS
- Database: PostgreSQL
- Deploy: Vercel + Railway

Teslim kriteri:
- Demo URL
- GitHub repo
- Test
- Sprint raporu

Sure: 5 gun

Butce limiti: Dusuk

Notlar: Ilk versiyon sade olsun, kurumsal satis demosu icin yeterli olsun.
```
