# 30 Day Plan

## Hafta 1: Temel Isletim Sistemi

Hedef:
- Agent Office'in calisma kurallarini netlestirmek.
- Sprint girisi ve GitHub akisini kurmak.

Isler:
- Telegram bot MVP.
- GitHub repo ve issue sablonlari.
- Sprint plan parser.
- Agent rol promptlari.
- Quality gate dokumani.

Cikti:
- Telefonda sprint gonderilebilir.
- Sprint GitHub issue'larina donusur.

## Hafta 2: Kod Uretim ve PR Akisi

Hedef:
- Agentlarin branch acip PR uretebildigi ilk is akisini kurmak.

Isler:
- Frontend/backend starter template.
- GitHub Actions.
- Test komutlari.
- PR sablonu.
- Demo deploy pipeline.

Cikti:
- Basit bir MVP otomatik PR ve demo olarak cikar.

## Hafta 3: QA, Raporlama ve Musteri Teslimati

Hedef:
- "Calisti" demeden once gercek kontrol yapmak.

Isler:
- Playwright smoke testleri.
- Sprint raporu generator.
- Demo kontrol checklist.
- Security checklist.
- Musteri teslim paketi.

Cikti:
- Her sprint sonunda demo linki ve rapor uretilir.

## Hafta 4: Pazar Arastirma ve Satis Paketi

Hedef:
- Agent Office'i musteri alabilecek bir hizmete donusturmek.

Isler:
- Turkiye pazar arastirma dongusu.
- Ilk 10 urun fikri.
- Kurumsal teklif PDF'i.
- LinkedIn/e-posta satis metinleri.
- 10 gunluk MVP Sprint paketi.

Cikti:
- Ilk kurumsal gorusmeler icin teklif ve demo hazirdir.

## 30 Gun Sonu Basari Kriteri

- Telefonda sprint gonderebiliyorsun.
- Agentlar GitHub issue/PR akisi ile calisiyor.
- CI test kapisi var.
- Demo deploy ediliyor.
- Sprint raporu uretiliyor.
- Yeni urun fikirleri otomatik arastiriliyor.
- Kurumsal firmaya sunulacak paket hazir.
