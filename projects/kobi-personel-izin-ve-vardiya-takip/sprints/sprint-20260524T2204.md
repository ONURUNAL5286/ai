# sprint-20260524T2204 - KOBI Servis Operasyon Merkezi - Futuristic UI ve Kalici Kayit

## Goal

Mevcut servis operasyon paneline tarayici icinde kalici veri saklama ve daha modern, futuristic, demo etkisi yuksek bir arayuz tasarimi eklemek.

## Target User

Teknik servis firmalari, saha operasyon yoneticileri, KOBI sahipleri

## Main Features

- Is emirleri localStorage kullanilarak tarayicida kalici tutulmali
- Teknisyen kayitlari localStorage kullanilarak tarayicida kalici tutulmali
- Yeni is emri eklendiginde sayfa yenilense bile kayit kaybolmamali
- Yeni teknisyen ekleme formu olmali
- Is emri durum guncelleme alani olmali
- Teknisyen atama degistirme alani olmali
- Demo verileri ilk acilista localStorage bos ise otomatik yuklenmeli
- Demo verileri sifirla butonu olmali
- Futuristic koyu tema uygulanmali
- Neon vurgulu metrik kartlari eklenmeli
- Cam efekti benzeri panel tasarimi uygulanmali
- Durum etiketleri renkli ve belirgin olmali
- Acil ve geciken is emirleri icin parlak uyari vurgulari olmali
- Arama ve filtreler localStorage verisi uzerinden calismali
- Mobil ekranda futuristic tasarim bozulmadan tek sutuna dusmeli

## Must Haves

- Mevcut proje klasoru guncellenmeli, yeni proje acilmamali
- public/index.html gercek uygulama ekranini korumali
- Sadece DONE listesi yazan ekran kabul edilmez
- Kayitlar sayfa yenilendikten sonra kaybolmamali
- localStorage kullanimi kodda net gorunmeli
- Futuristic UI belirgin sekilde gorunmeli
- Harici npm paketi eklenmemeli
- Harici CSS framework kullanilmamali
- start.cmd calismaya devam etmeli
- README ve STATUS.md guncellenmeli
