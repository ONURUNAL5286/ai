# sprint-20260524T1912 - KOBI Stok ve Siparis Takip

## Goal

Kucuk isletmelerin stok, siparis ve kritik stok uyarilarini tek panelden takip ettigi web uygulamasi.

## Target User

KOBI sahipleri, depo sorumlulari, satis ekipleri

## Main Features

- Urun kaydi, kategori, stok adedi ve alis-satis fiyati ekleme
- Siparis olusturma ve siparis durumlarini takip etme
- Kritik stok seviyesine dusen urunleri uyarili gosterme
- Gunluk satis, toplam stok degeri ve bekleyen siparis metrikleri
- Urun ve siparis listelerinde arama ve filtreleme

## Must Haves

- Mobil uyumlu sade panel
- Demo verilerle ilk acilista kullanilabilir ekran
- Calisan start.cmd
- README kurulum adimlari
- Smoke test checklisti
